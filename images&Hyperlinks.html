Images

we can display images both online(by copying image link) and offline.Image is unpaired Tag

Mandatory Attributes of Img tag

  • src=> source of image (location of image)
  • alt=> alternative text(alternative to image)
  • title=> tool tip for image
  • height=> to increase/decrease height of image
  • width=> to increase/decrease width of image

Types of images

  1. jpg=>Joint Photographic group
  2. png=>Portable Network graphics*
  3. webp=>Webpage image*
  4. jpeg=>Joint Photographic expert group
  5. gif=>Graphical Interchange format
  6. psd=>Photosjop document
  7. svg=>Scalable Vector graphics*
